## How to Install
**Setup Version** ( This will run automatically on startup )
* Download Setup
* **Uncheck Synapse to Run on Startup** (Razer Central Settings Panel)
* Install `SynapseKillerSetup.exe`

**Portable Version**
* Download Portable
* Run `SynapseKiller.exe`
